npm安装

前往Node.js官网选择合适版本

Hexo环境配置

npm install hexo-cli -g

hexo init blog

cd blog

npm install

hexo server

Hexo基本用法

hexo server # 本地模拟

hexo new "new post" # 新建一篇文章

hexo clean # 清空缓存

修改主题

取决于你自己。

本博客主题Minimalism。

修改配置文件后，务必清除缓存使配置生效

npm run clean

部署到Github

注意您上传的项目名称应为username.github.io

  1. hexo-deployer-git插件
npm install --save hexo-deployer-git
  1. 修改_config.yml文件
deploy:
  type: git
  repo: xxx.git
  branch: page # 部署到page分支
  1. 发布到github
hexo cl # clear cache
hexo g 	# 生成public
hexo d	# deploy
  1. 修改setting设置中的page中分支为上述分支。

开始你的✍吧
